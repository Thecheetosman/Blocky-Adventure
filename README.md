# 🟦 Blocky Adventure V28
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

A text-based, procedural exploration and survival game built entirely in HTML5 and JavaScript. Traverse a vast world, discover unique locations, and take to the skies in your airship.

**[🎮 Play Blocky Adventure Now!](https://thecheetosman.github.io/Blocky-Adventure/)**

---

## 🚀 Features
- **Vast Exploration:** A massive world map with 384 unique settlements and burgs to discover.
- **Airship System:** Travel faster across the world with the newly integrated airship mechanics.
- **Dynamic Questing:** Engage in quests and handle encounters at various locations.
- **Retro Aesthetic:** A clean, terminal-style interface for a classic gaming feel.
- **Built-in Cheats:** For developers and testers, use the integrated cheat console to unlock locations or summon resources.

## 🕹️ How to Play
1. **Movement:** Use the on-screen controls or keyboard shortcuts to navigate the map.
2. **Interact:** Click on settlements to explore them or initiate quests.
3. **Save/Load:** Your progress is tracked within the session; keep your tab open to maintain your journey!

## 🛠️ Installation & Development
Since this is a portable web game, there is no installation required. To run it locally:
1. Clone this repository: `git clone https://github.com/YOUR_USERNAME/blocky-adventure.git`
2. Open `index.html` in any modern web browser.
